Updating VerseOfTheDay File   
1. Find the new file [here](https://partner-support.youversion.com/l/en/article/2z6brl1tus-verse-of-the-day-calendar)   
2. Replace the file in the project folder   
3. Open it in Intellij (NOT Excel) and replace the first row with an entry matching the current day's verse   
4. Upload new file to file server
